# 카피라이팅 에이전트 (Copy Agent) — 정부 정책 홍보 버전

## 역할
리서치 결과와 리플릿 데이터를 바탕으로 10개 섹션의 정책 홍보 카피를 작성하고,
각 섹션에 들어갈 **AI 이미지 생성 프롬프트**를 함께 제공합니다.

## 입력
- `input/product_brief.json` — 정책 브리프
- `collected/product_data.json` — 수집된 정책 정보
- `research/research_report.md` — 리서치 리포트
- `장해진단서_제도_리플렛.pdf` — 공식 리플릿 원본 참조

## 상세페이지 구조 (정부 정책 홍보 10섹션)

### Section 01: Hero (정책 헤더)
```
목표: 핵심 변화를 한눈에 전달 + 신뢰감 확보

구성요소:
- headline: 정책 핵심 변화 한 줄 (3가지 옵션)
- subheadline: 시행일 + 대상 명시
- policy_badge: "2026.1월 시행" 뱃지
- authority_mark: "국가보훈부" 공식 마크
- cta_text: 행동 유도 (예: "내 근처 발급병원 찾기")

이미지 프롬프트:
- 메인 비주얼: "대한민국 태극기 모티프를 활용한 정부 정책 홍보 배너.
  중앙에 깨끗한 현대식 종합병원 건물, 앞에 밝은 표정의 중년 남성(보훈대상자)이
  가족과 함께 병원에 들어서는 모습. 따뜻한 아침 햇살, 희망적이고 밝은 분위기.
  한국 정부 공식 디자인 스타일, 파란색(#003478)과 빨간색(#C8102E) 포인트.
  깨끗하고 현대적인 일러스트 스타일. 16:9 비율."
```

### Section 02: Before/After (변화 대비)
```
목표: "이렇게 달라집니다" — 기존 vs 변경 후 극적 대비

구성요소:
- before_title: "기존" 라벨
- before_items: 기존의 불편 사항 3-4개
- after_title: "2026년 1월부터" 라벨
- after_items: 변경 후 편리해진 점 3-4개
- highlight: 핵심 변화 강조 문구

이미지 프롬프트:
- Before 이미지: "고령의 한국인 노인이 지팡이를 짚고 먼 거리의 큰 병원(보훈병원)으로
  힘겹게 이동하는 모습. 지도 위에 집에서 보훈병원까지 긴 빨간 점선 경로가 표시됨.
  회색 톤, 피곤하고 걱정스러운 분위기. 한국적 도시 배경. 일러스트 스타일."
- After 이미지: "밝은 표정의 한국인 중년 남성이 동네 가까운 종합병원에 편안하게
  걸어 들어가는 모습. 지도 위에 집에서 가까운 병원까지 짧은 초록색 경로 표시.
  밝은 톤, 편안하고 안심되는 분위기. 꽃이 피어있는 봄날 배경. 일러스트 스타일."
```

### Section 03: Policy Overview (정책 소개)
```
목표: 제도를 명확하고 쉽게 설명

구성요소:
- intro: 제도 배경 설명 (1-2문장)
- legal_basis: 법적 근거 (개정 국가유공자법 '23.6.17. 시행)
- key_changes: 핵심 변화 내용 (발급기관 49곳→140곳)
- institution_breakdown: 기관별 분류
  - 상급종합병원 47곳
  - 보훈병원 5곳
  - 종합병원급 위탁병원 86곳
  - 국군수도병원 1곳 + 경찰병원 1곳
- what_is_it: 장해진단서란 무엇인지 쉬운 설명

이미지 프롬프트:
- 인포그래픽: "숫자 49에서 140으로 변화하는 다이내믹한 인포그래픽.
  49는 작고 회색, 140은 크고 파란색(#003478)으로 강조.
  주변에 병원 아이콘들이 점점 늘어나는 시각적 효과.
  깔끔한 한국 정부 보고서 스타일. 화이트 배경. 플랫 디자인."
- 기관분류 다이어그램: "원형 차트 또는 트리 구조로 140곳 의료기관 분류를 보여주는
  인포그래픽. 상급종합병원(47), 보훈병원(5), 위탁병원(86), 기타(2).
  각 카테고리별로 다른 색상 사용. 깔끔한 플랫 디자인."
```

### Section 04: How It Works (이용 절차)
```
목표: 신청부터 심사까지 3단계를 쉽고 명확하게

구성요소:
- steps: 3단계 프로세스
  1. 관할 보훈관서에서 대상 여부 확인
  2. 가까운 발급병원 방문하여 장해진단서 발급
  3. 장해진단서 제출 → 보훈심사위원회 상이등급 심사
- each_step: { number, title, description, tip }
- important_note: "보훈병원 신체검사를 생략할 수 있습니다"

이미지 프롬프트:
- 스텝 다이어그램: "3단계 수평 플로우차트. 각 단계가 둥근 사각형으로 표현.
  1단계: 전화기 아이콘 + '보훈관서 확인', 2단계: 병원 아이콘 + '장해진단서 발급',
  3단계: 서류 아이콘 + '보훈심사위원회 심사'. 단계 사이에 화살표 연결.
  단계별로 파란색 그라데이션 적용. 깔끔한 플랫 디자인. 한국어 텍스트 포함."
- 각 단계별 일러스트:
  - Step 1: "한국인 중년 남성이 전화로 보훈관서에 문의하는 모습. 따뜻한 실내,
    편안한 표정. 말풍선에 '대상 여부 확인' 텍스트. 밝은 일러스트 스타일."
  - Step 2: "한국인 보훈대상자가 동네 종합병원 접수창구에서 장해진단서 발급을
    신청하는 모습. 친절한 직원이 안내. 밝고 깨끗한 병원 로비. 일러스트 스타일."
  - Step 3: "서류가 보훈심사위원회로 전달되어 심사가 진행되는 모습.
    '신체검사 PASS' 도장이 찍힌 서류. 안도하는 보훈대상자. 일러스트 스타일."
```

### Section 05: Benefits (핵심 혜택)
```
목표: 제도 활용의 구체적 이점을 체감시키기

구성요소:
- headline: "이렇게 좋아집니다"
- benefits: 핵심 혜택 리스트 4-5개
  - 접근성: 집 근처에서 편리하게 발급
  - 시간 절감: 보훈병원 방문 및 대기 시간 불필요
  - 절차 단축: 신체검사 생략으로 등급심사 기간 단축
  - 전국 커버리지: 전국 140곳에서 발급 가능
  - 동일한 심사 품질: 보훈심사위원회 최종 등급심사는 동일

이미지 프롬프트:
- 혜택 아이콘 세트: "5개의 원형 아이콘 세트. 각각: (1) 집+병원 근접 아이콘,
  (2) 시계+체크마크 아이콘, (3) 서류 간소화 아이콘, (4) 대한민국 지도+핀 아이콘,
  (5) 저울(공정성) 아이콘. 파란색(#003478) 모노톤 플랫 디자인."
- 비교 인포그래픽: "기존 vs 변경 소요시간 비교 바 차트.
  기존: 보훈병원 방문(긴 막대, 빨간색), 변경: 가까운 병원(짧은 막대, 파란색).
  시간 절감량을 화살표로 강조. 깔끔한 인포그래픽 스타일."
```

  상급종합병원은 파란 핀, 보훈병원은 빨간 핀, 위탁병원은 초록 핀으로 구분.
  수도권 확대 삽입 지도 포함. 깔끔한 벡터 스타일. 범례(legend) 포함."

```

  - 고엽제후유증: 검진에서 인정받은 고엽제후유증 질병

  - 고엽제후유의증
  - 고엽제후유증 2세 환자
  - 생활능력이 없는 정도의 장애판정

- choice_note: "본인 선택 사항 — 기존 보훈병원 신체검사도 계속 이용 가능"

이미지 프롬프트:
- 대상자 구분 인포그래픽: "2열 레이아웃. 왼쪽: 초록색 체크마크와 함께
  이용 가능 대상 리스트. 오른쪽: 빨간 X마크와 함께 이용 불가 대상 리스트.
  깔끔한 카드 디자인. 아이콘과 텍스트 조합. 정부 공식 디자인 스타일."
- 안내 캐릭터: "친근한 한국 정부 캐릭터(심플 일러스트)가 손가락으로
  체크리스트를 가리키며 안내하는 모습. 밝고 친근한 톤."

### Section 09: Required Documents (필요서류/검사항목)
```

  깔끔하게 정리된 모습. 펜과 청진기가 옆에 놓여있음.
  깨끗하고 전문적인 의료 일러스트 스타일. 파란색 톤."
