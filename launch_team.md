## 사용법

Claude Code에서 아래 프롬프트를 입력하세요:

### 기본 실행
```
상세페이지 에이전트 팀을 구성해서 다음 정책의 홍보 상세페이지를 만들어줘.

정책: 국가보훈 장해진단서 발급기관 확대 (2026.1월 시행)
목적: 보훈대상자가 보훈병원 대신 집 근처 가까운 종합병원(위탁병원)에서 장해진단서를 발급받도록 적극 유도
참고 자료: 장해진단서_제도_리플렛.pdf
이미지: AI 이미지 생성 프롬프트 포함

팀 구성:
1. collector 에이전트로 정책 정보 수집 (리플릿 PDF 데이터 추출 포함)
2. researcher 에이전트로 보훈정책/대상자/SEO 분석
3. copywriter 에이전트로 10개 섹션별 카피 작성 + 이미지 프롬프트 포함
4. designer 에이전트로 HTML 디자인 시안 제작 (AI 이미지 프롬프트 반영)
5. prompt-engineer 에이전트로 품질 검수

작업 폴더: detail-page-project/
```

### 이미지 생성 포함 실행
```
위 상세페이지를 만들되, 각 섹션에 생생한 이미지를 포함해줘.
Canva MCP 연동이 가능하면 Canva로 이미지를 생성하고,
불가능하면 각 섹션별 AI 이미지 생성 프롬프트를 design/ 폴더에 저장해줘.

이미지 스타일 가이드:
- 한국 정부 공식 디자인 톤 (파란색 #003478, 빨간색 #C8102E 포인트)
- 따뜻하고 신뢰감 있는 일러스트 스타일
- 한국적 요소 반영 (태극기 모티프, 무궁화 등)
- 정보 전달 중심의 인포그래픽 활용
- 고령자도 읽기 편한 큰 텍스트와 명확한 대비
```

### 섹션 구조 (정부 정책 홍보 10섹션)
```
1. Hero (정책 헤더) — 핵심 변화 한 줄 + 시행일
2. Before/After (변화 대비) — 기존 vs 변경 후 비교
3. Policy Overview (정책 소개) — 제도 배경, 핵심 변화
4. How It Works (이용 절차) — 3단계 안내
5. Benefits (핵심 혜택) — 접근성, 시간 절감 등
6. Hospital Map (발급기관 안내) — 전국 140곳
7. Target Audience (대상자 안내) — 이용 가능/불가 구분
8. FAQ (자주 묻는 질문) — 실제 민원 기반
9. Required Documents (필요서류) — 진단항목 요약
10. CTA (행동 유도) — 보훈관서 연락처 + 병원 찾기
```
